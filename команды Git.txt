git add . - добавить все файлы
git commit -m 'название того что добавляем' - добавить коммит
git push - добавить коммиты в git hub
git push origin название ветки
git log - список коммитов
git branch - актуальная ветка (+список вток)
git checkout  - переключиться на ветку
git push --set-upstream origin и название ветки - добавить ветку в удаленный репозиторий
